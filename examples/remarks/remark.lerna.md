# lerna 项目使用

## lerna 安装

node.js 版本 `Node >= 16.0.0`。

```bash
yarn global add lerna
```

### lerna 初始化项目

```bash
lerna init --independent
```

### lerna 创建 npm 包

```bash
lerna create @magic-tiger/common-ui
```
